# Vector Forge Memory

## User coding conventions
- Lazarus/FPC naming: lowerCamelCase
- local variable prefix: p
- field prefix: f
- global prefix: g
- parameter prefix: a
- optional type letters:
  - s = string
  - i = numeric
  - b = boolean
- English names only

## Operating principles
- Return working solutions ready for deployment when possible.
- Do not change public APIs unless explicitly allowed.
- Record important decisions after each completed task.
